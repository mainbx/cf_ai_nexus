📬 Gmail Integration Setup

Connect your Gmail to import emails directly into your personal wiki. The AI will organize and summarize them for you.

1

Create a Google Cloud Project

Go to console.cloud.google.com and create a new project (it's free).

  1. Click "Select a project" at the top, then "New Project"
  2. Name it anything (e.g., "Nexus Wiki")
  3. Click "Create"
2

Enable the Gmail API

  1. In your project, go to APIs & Services > Library
  2. Search for "Gmail API"
  3. Click on it and press Enable
3

Configure OAuth Consent Screen

  1. Go to APIs & Services > OAuth consent screen
  2. Select External user type
  3. Fill in the app name (e.g., "Nexus Wiki") and your email
  4. Add the scope: gmail.readonly
  5. Under Test users, add your own Gmail address
  6. Save and continue through each step
⚠️ Keep the app in "Testing" mode — no verification needed for personal use!
4

Create OAuth Credentials

  1. Go to APIs & Services > Credentials
  2. Click Create Credentials > OAuth client ID
  3. Application type: Web application
  4. Add authorized redirect URI: http://localhost:8787/api/gmail/callback
  5. Click Create
  6. Copy the Client ID and Client Secret
5

Add Credentials to Nexus

Add these to your .dev.vars file (local) or as Cloudflare secrets (deployed):

GOOGLE_CLIENT_ID=your-client-id-here
GOOGLE_CLIENT_SECRET=your-client-secret-here

Then restart the dev server.

6

Connect & Import

Once credentials are set up:

You'll be redirected to Google to authorize. After authorizing, tell the AI: "Import my recent emails"

API / MCP Endpoint

Nexus exposes a REST API that external AI agents, tools, or scripts can use to read, search, and manage your wiki data programmatically.

Base URL


          


          

            
Available Endpoints

            

              
                
MethodEndpointDescription

              
              
                
GET/mcpService discovery — lists all endpoints

                
POST/mcp/searchSemantic search: {"query": "..."}

                
GET/mcp/entriesList entries: ?type=contact&limit=10

                
GET/mcp/entries/:idGet a single entry by ID

                
POST/mcp/entriesCreate entry: {"type","title","content"}

                
PUT/mcp/entries/:idUpdate an entry

                
DELETE/mcp/entries/:idDelete an entry

                
GET/mcp/typesList entry types with counts

                
POST/mcp/askAsk AI a question: {"question": "..."}

              
            

          


          

            
Example: Search your wiki

            
curl -X POST /mcp/search \
  -H "Content-Type: application/json" \
  -d '{"query": "John email address"}'

          


          

            
Example: Create an entry

            
curl -X POST /mcp/entries \
  -H "Content-Type: application/json" \
  -d '{"type":"contact","title":"Jane Doe","content":"{\"name\":\"Jane\",\"email\":\"jane@example.com\"}"}'
